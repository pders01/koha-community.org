# Report a Problem

**If it is a security bug, [report it here](http://bugs.koha-community.org/bugzilla3/enter_bug.cgi?product=Koha%20security)**

Otherwise:

**You didn’t find your issue listed?** [Make yourself an account](https://bugs.koha-community.org/bugzilla3/createaccount.cgi), and then [create a new bug report!](http://bugs.koha-community.org/bugzilla3/enter_bug.cgi)

**If you’re not sure about what to type in all of those boxes,** you can check out our [Bug Reporting Guidelines.](https://wiki.koha-community.org/wiki/Bug_Reporting_Guidelines)

If you find yourself here a lot, you can [install a Koha Bugzilla Quicksearch function](javascript:window.external.AddSearchProvider('https://bugs.koha-community.org/bugzilla3/search_plugin.cgi')) in Firefox or Internet Explorer 7+, and search our bug database directly from your browser quick search bar. **Note:** If you had previously installed the browser search from our old Bugzilla at [http://bugs.koha.org](http://bugs.koha.org), [you will need to remove it](http://www.ehow.com/how_5666777_remove-engine-firefox_s-search-bar.html) before you add the new one.
