# Other Docs

Koha libraries have been creating their own documentation for their staffs and sharing it online. This page lists all known available documentation. If you know of some documentation that is missing from this list, please email the [koha or koha-docs mailing lists](https://koha-community.org/support/koha-mailing-lists/).
